## 📋 Prerequisites

Before starting this program, you should be comfortable with:

- **Python 3.10+** — functions, classes, list comprehensions, decorators, virtual environments
- **Basic statistics** — mean, median, standard deviation, distributions, p-values
- **Git & GitHub** — clone, commit, push, pull, branching, pull requests
- **Command line** — navigating directories, running scripts, installing packages with `pip`
- **Jupyter Notebooks** — running cells, markdown, basic plotting
- **Basic SQL** — SELECT, WHERE, GROUP BY (helpful but not mandatory)

> 💡 If you're rusty on any of these, spend 1–2 days brushing up before starting Task 01. The [Python Data Science Handbook](https://jakevdp.github.io/PythonDataScienceHandbook/) is a great refresher.

## ⚙️ Setup

```bash
git clone https://github.com/musagithub1/ML-Internship-Tasks.git
cd ML-Internship-Tasks
python -m venv .venv
source .venv/bin/activate    # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 🪟 Windows Notes

- Use **PowerShell** or **Git Bash** (not `cmd.exe`).
- Activate the virtual environment with: `.venv\Scripts\activate`
- If `pip install` fails for PyTorch, install it separately:
  ```bash
  pip install torch torchvision --index-url https://download.pytorch.org/whl/cpu
  ```

### 🍎 macOS Notes

- Install Python via [Homebrew](https://brew.sh): `brew install python@3.11`
- For Apple Silicon (M1/M2/M3), PyTorch has native MPS support — no extra steps needed.

### 🐳 Docker Setup

Docker is required for **Tasks 16, 22, and 24**. Install Docker Desktop before you reach Month 2:

- **Linux:** [Install Docker Engine](https://docs.docker.com/engine/install/)
- **macOS:** [Docker Desktop for Mac](https://docs.docker.com/desktop/install/mac-install/)
- **Windows:** [Docker Desktop for Windows](https://docs.docker.com/desktop/install/windows-install/) (requires WSL 2)

Verify your installation:
```bash
docker --version
docker run hello-world
```
