## 🧭 How to Navigate

Each task lives in its own folder under the relevant month, e.g.:

```
Month_1_Foundations/Task_01_Exploratory_Data_Analysis_Deep_Dive/
├── README.md                 ← start here
├── TASK.md                   ← full requirements
├── RESOURCES.md              ← curated learning materials
├── SUBMISSION_GUIDELINES.md  ← how to submit
├── EVALUATION_RUBRIC.md      ← how you're graded
├── BONUS_CHALLENGE.md        ← stretch goal
├── STARTER_FILES/            ← scaffolding code
├── DATASETS/                 ← dataset info + .gitignore
├── REFERENCES/               ← your personal reading log
└── EXAMPLES/                 ← target quality bar
```

## 🛠️ Tech Stack You'll Use

- **Languages:** Python 3.10+
- **Core libs:** NumPy, Pandas, Scikit-learn, Matplotlib, Seaborn
- **ML/DL:** XGBoost, LightGBM, CatBoost, PyTorch, Hugging Face
- **MLOps:** MLflow, DVC, Docker, FastAPI, GitHub Actions
- **Interpretability:** SHAP, LIME, Evidently AI
- **Deployment:** Render, Hugging Face Spaces, Streamlit

## ⚙️ Setup

```bash
git clone https://github.com/musagithub1/ML-Internship-Tasks.git
cd ML-Internship-Tasks
python -m venv .venv
source .venv/bin/activate    # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## 📏 Internship Rules

1. Every task **must** be pushed to GitHub.
2. Every task **must** have a written report.
3. Every commit message follows [Conventional Commits](https://www.conventionalcommits.org/).
4. No commits with credentials, raw datasets >100MB, or plagiarized notebooks.
5. Submit by opening a GitHub Issue titled `Review: Task NN`.
