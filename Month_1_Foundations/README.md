# 📊 Month 1 — Advanced Data Analysis & ML Foundations

> **Weeks 1–4** | Build a rock-solid foundation in data analysis, statistical thinking, and core ML algorithms. Every task here teaches a skill you'll use in every task that follows.

---

## 🎯 Learning Objectives

By the end of Month 1, you will be able to:

1. **Conduct thorough exploratory data analysis** — uncover patterns, outliers, and distributions in any tabular dataset
2. **Clean and preprocess messy real-world data** — handle missing values, duplicates, encoding, and scaling at production quality
3. **Engineer features that improve model performance** — create interaction terms, aggregations, temporal features, and domain-specific signals
4. **Apply statistical hypothesis testing** — choose the right test (t-test, chi-squared, ANOVA), interpret p-values, and avoid common pitfalls
5. **Build and evaluate regression models** — from a hand-coded linear regression to regularized models, with proper train/test discipline
6. **Compare classification algorithms rigorously** — logistic regression, decision trees, SVM, KNN — and explain when each works best
7. **Perform unsupervised learning for business segmentation** — K-Means, DBSCAN, hierarchical clustering, with silhouette and elbow analysis
8. **Design robust evaluation strategies** — cross-validation, stratification, leakage prevention, and metric selection for imbalanced data
